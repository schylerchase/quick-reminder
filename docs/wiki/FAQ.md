# FAQ

## Is Quick Reminder a replacement for the Tasks plugin?

No. Quick Reminder focuses on natural-language reminders, local scheduling, and a lightweight Markdown dashboard. It can optionally integrate with the Tasks plugin.

## Does it work on mobile?

Yes. It is not desktop-only. Mobile uses Obsidian in-app notices rather than reliable background push notifications.

## Will reminders fire while Obsidian is closed?

Not at the exact due time. Quick Reminder runs inside Obsidian. Missed reminders can be shown on next launch.

## Why is Add reminder disabled?

The task needs a future date phrase, such as `tomorrow 9am` or `Friday 2pm`.

## Can I hide a task without deleting it?

Yes. Use **Ignore**. The task remains in the source note.

## Can I replace the starter dashboard?

Yes. The starter dashboard is meant to be edited, renamed, or deleted. It is a starting board, not a permanent cage.

## Where are reminders stored?

Reminder state is stored in plugin data. The optional Markdown mirror is generated output.
