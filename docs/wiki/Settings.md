# Settings

## Reminder settings

| Setting | Purpose |
|---|---|
| Mirror to markdown | Keeps a generated reminders note in the vault |
| Mirror file path | Controls where the generated reminder note lives |
| Default snooze | Controls the manager snooze button duration |
| Fire missed reminders on launch | Shows overdue reminders when Obsidian reopens |
| Sound on notify | Enables notification sound where available |

## Task dashboard settings

| Setting | Purpose |
|---|---|
| Reveal active file in file explorer | Expands and highlights the active note |
| Tasks plugin integration | Uses the Tasks plugin modal when available |
| Task section headings | Headings used by insert/template actions |
| Auto-insert task sections | Adds sections to empty new notes in selected folders |
| Auto-insert managed block | Adds managed task blocks to selected folders on open |

## Starter dashboard settings

| Setting | Purpose |
|---|---|
| Starter board path | The vault-relative file created by Start with template |
| Create/open starter board | Creates or opens the replaceable starter dashboard |

The starter board path must end in `.md`. Existing files are opened and never overwritten.
